# Turf Booking System

## Overview
The **Turf Booking System** is a full-stack web application that allows users to view available turfs, check time slots, and book turf reservations online.  
It is built using **React.js** for the frontend, **Spring Boot** for the backend, and **MySQL** as the database.

The system is designed to simplify turf management and booking operations with a clean and responsive user interface.

---

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- CSS
- Axios

### Backend
- Spring Boot
- Java
- REST APIs

### Database
- MySQL

---

## Features

### User Features
- View available turfs
- Select date and time slots
- Book turf
- View booking history

### Admin Features
- Add / update turf details
- Manage bookings
- View all reservations
